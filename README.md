# Referease – Trust-Based Job Referral Platform

Referease is a high-fidelity, scalable digital platform designed to connect job seekers with industry professionals for seamless, privacy-conscious referrals.

## 🚀 Key Features

- **Match Discovery**: AI-driven connection suggestions for candidates and employees.
- **Flash Chat**: A time-limited (5-minute) communication bridge between seekers and referrers with built-in moderation to prevent platform bypass.
- **Monetization Engine**: Integrated ₹100 Top-Up system for chat extensions and ₹1000/mo Premium tiers for employers.
- **Anti-Spam & Trust Moderation**: Automated profile suspension for low-engagement/high-rejection employers.
- **Privacy-First Design**: Selective identity masking and structured referral requests.

## 🛠️ Technology Stack

- **Frontend**: React.js, Vite, Vanilla CSS, Lucide-React Icons.
- **Backend (Mock)**: Node.js, Express.

## 💻 Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation & Development

1. **Clone the repository** (or download the folder)
2. **Install dependencies**:
   ```bash
   npm install
3. **Start the Frontend**:
   ```bash
   npm run dev
   ```
4. **Start the Backend (Optional for mock APIs)**:
   ```bash
   node server.js
   ```

The application will be available at `http://localhost:5173`.
